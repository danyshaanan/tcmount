
  Usage: BIN [[-vhl] | [-u] <name> | <file> <dir> | -u <file|dir>]

  Options:
    -v    Output the version number
    -h    Output this help message
    -l    List mounted volumes
    -u    Unmount pair

  Examples:
    Use "BIN" to list currently mounted volumes
    Use "BIN media" to mount ~/.tc/media.tc onto ~/media
    Use "BIN -u media" to unmount it ~/media
    Use "BIN ~/anytcfile ~/any/folder" to mount outside the convention
    Use "BIN -u ~/anytcfile" or "BIN -u ~/any/folder" to unmount outside the convention

  Reserved names:
RESERVED
